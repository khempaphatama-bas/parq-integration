Folder rules:
- Keep only source/reference artifacts here.
- Derived analysis should live in `02_Discovery` or `03_Architecture`.
- When a source is superseded, keep the file but mark its status in `MASTER_INDEX.md`.

- `Domain_References/` - Workplace and Parking source/reference documents.
- `API_and_System_References/` - OBK, IAM, SSO, CMS, notification, profile, OTP, and related system reference documents.
